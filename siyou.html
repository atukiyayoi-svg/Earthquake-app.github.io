EPICA 仕様書及び取扱説明書

バージョン: V4.2.0

1. アプリケーションの使い方

本システム「EPICA」は、各種APIから取得したデータを基に、リアルタイムの地震動（強震モニタ）、緊急地震速報（EEW）、津波情報、各種気象情報を2Dマップ上で総合的に監視・可視化するツールです。

基本操作

  • 起動と音声: 起動画面で「スタート」をクリックするとシステムが初期化されます。ブラウザの仕様上、画面内を一度クリック/タップすることで音声（読み上げ・通知音）が有効になります。
  • 表示モードの切替: 画面中央下部のボタンでモードを切り替えます。
    • 通常表示 最新の地震情報、過去の震源地、予測震度マップなどを総合的に表示します。
    • リアルタイム揺れのみ 過去の地震のアイコンや色塗りを非表示にし、現在起きているリアルタイムの揺れ（強震モニタの点）のみにフォーカスします。
  • 設定 (⚙): 右下の設定ボタンから、各種音量（基本音量、BGM、各種SE）、音声読み上げの速度・ピッチ、描画オプション（EEW予測マップ、観測点表示など）を変更できます。
  • 履歴の確認: 右側の「地震情報ウィンドウ」から過去の地震をクリックすると、その地震の詳細（震央、最大震度、観測点ごとの震度）がマップおよび左パネルに表示されます。履歴リストはスクロール可能で、ウィンドウ内に6件分が表示され、快適に過去情報を遡ることができます。
  • 自由付随コメントの確認: 右側の「自由付随コメント」パネルにて、緊急地震速報（EEW）受信時の補足事項や、地震情報に含まれる付随コメントを確認できます。EEWの補足事項はオレンジ色で優先的に強調表示されます。
  • 速報情報の表示: 地震発生直後に気象庁から発表される「震度速報」および「震源に関する情報」を受信した際、これらの暫定情報は履歴リストには追加せず、マップ上への震度観測点・震央マーカーの即時描画、および左パネルでの情報更新をリアルタイムに行います。確定情報を受信するまでマップの塗分け状態を維持し、震央マーカーは他の要素に隠れないよう最前面に強調表示されます。

⚠️ 予測機能に関する重要・免責事項

本システム上で描画される以下の情報は、独自のアルゴリズムに基づく完全な「予測」ならびに「推定」であり、正確なものではありません。

  • P波・S波の伝播サークルの描画: 理論値に基づく簡易的な描画であり、実際の地震波の到達を示すものではありません。
  • J-SHISモデルを用いた「予測震度マップ」: 地盤増幅率（ARV）を用いた計算上の予測であり、実際の観測震度を保証するものではありません。
  • 予測長周期地震動階級 (LPGM): マグニチュードと深さから簡易算出された参考値です。

本システムの情報を用いて生じた、いかなる損害・トラブルについても作者は一切の責任を負いません。実際の避難行動や意思決定の際は、必ず気象庁や自治体が発表する公式の防災情報に従ってください。

2. 取得情報の情報元・使用API

本システムは以下の外部APIおよびオープンデータを利用して構築されています。

  • リアルタイム震度データ (強震モニタ):
    取得元: Yahoo! 防災速報 / NIED (防災科学技術研究所)
    用途: リアルタイムの揺れの可視化。
  • 地震履歴・津波情報 (P2Pquake API):
    取得元: P2P地震情報 API v2 (https://www.p2pquake.net/json_api_v2/)
    用途: 過去の地震履歴一覧の取得、震度速報・震源情報の受信、津波警報・注意報のエリア情報の取得。
  • 緊急地震速報 (Wolfx API):
    取得元: Wolfx WebSocket API (https://wolfx.jp/)
    用途: 緊急地震速報（警報・予報）の即時受信および表示。
  • 総合防災情報 (AXIS API):
    取得元: AXIS API (Prioris)
    用途: 気象情報、火山情報、ニュース速報の受信、および緊急地震速報（予報・警報）の受信。※利用にはユーザー自身が取得した「アクセストークン」をシステム設定に入力する必要があります。設定からEEWの受信元としてWolfxまたはAXISを選択可能です。
  • 表層地盤増幅率データ (J-SHIS API):
    取得元: 防災科学技術研究所 J-SHIS API V4 (https://www.j-shis.bosai.go.jp/api)
    用途: EEW受信時における、各地域（250mメッシュ相当）の精緻な予測震度の計算補正。
  • ベースマップタイル:
    取得元: CartoDB Dark Matter (https://carto.com/) / レンダリングエンジン: Leaflet.js

⚖️ 法律関連の注意事項 (気象業務法について)

日本国内において、気象庁以外の者が独自の計算式を用いて「予測震度」や「緊急地震速報に類する情報」を作成し、これを不特定多数の一般大衆に向けて配信・通知する行為（インターネット上での公開配信やSNSでの自動発信など）は、気象業務法に基づく気象庁長官の「予報業務許可（地震動）」が必要となる場合があります。

本システム（HTMLファイル）をユーザー自身がローカル環境等で実行し、自己の判断・監視のために個人的に利用する範囲（自己完結型の処理）であれば問題ありません。しかし、本システムの画面をライブ配信したり、生成された予測震度マップを不特定多数へ二次配信したりする行為は、関係法令に抵触する恐れがあるため十分にご注意ください。

※詳細については気象庁の公式見解をご確認ください。
